# MotorPH Basic Payroll System - Milestone 2

## Program Details
This is a basic Java-based payroll system developed for MotorPH. The program reads employee information and attendance records from a CSV/text file using Java file handling. 

It calculates the total hours worked per cutoff (strictly observing the 8:00 AM to 5:00 PM company policy) and computes the semi-monthly gross salary based on the employee's hourly rate. Furthermore, it applies government deductions (SSS, PhilHealth, Pag-IBIG, and Withholding Tax) exclusively to the 2nd cutoff salary based on the monthly gross salary. The entire system is built using purely procedural programming methods without utilizing Object-Oriented Programming (OOP) concepts, adhering to the project requirements.
